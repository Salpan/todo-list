# Todo List

**Минималистичное приложение для управления задачами**  
Проект создан для демонстрации навыков работы с **React**, **TypeScript** и современными инструментами сборки.  
Функционал включает: добавление, редактирование, удаление и фильтрацию задач с сохранением состояния в `localStorage`.

## Возможности

-   Добавление новых задач
-   Редактирование существующих задач
-   Удаление задач
-   Отметка выполнения (checkbox)
-   Фильтрация: все / активные / выполненные
-   Подсчёт количества невыполненных задач
-   Сохранение списка задач между обновлениями страницы

## Стек технологий

-   **React** — библиотека для построения интерфейсов
-   **Vite** — быстрый и современный сборщик
-   **TypeScript** — строгая типизация для надёжного кода
-   **React Hooks** (`useState`, `useEffect`, `useCallback`, `useMemo`) — управление состоянием и оптимизация рендеринга

## Установка и запуск

1. **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/Salpan/todo-list.git
    ```
2. **Перейдите в директорию проекта:**
    ```bash
    cd todo-list
    ```
3. **Установите зависимости:**
    ```bash
    npm install
    ```
4. **Запустите приложение:**
    ```bash
    npm run dev
    ```
    Приложение будет доступно по адресу: `http://localhost:5173`
